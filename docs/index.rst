PBRecipe
========

**PBRecipe** is a desktop recipe management application built with Qt (PySide6).
It stores recipes, ingredients, categories, techniques and units in a relational
database (SQLite, MariaDB or PostgreSQL) and can generate a self-contained PHP
web export for publication on a local or remote server.
